# ✈️ Aircraft Checklists

A collection of aircraft checklists formatted as Markdown files — useful for study, training, and simulation purposes and direct import to Apple Notes.

## 📥 Importing Checklists into Apple Notes on Mac

You can import these Markdown files into Apple Notes for easy offline access on your Mac, iPhone, or iPad:

In **Apple Notes** on your Mac select File -> Import Markdown -> navigate to the checklist `.md` file you want to import.

## ⚠️ Disclaimer

**These checklists are for educational and simulation purposes only.**

The checklists provided in this repository are intended to support learning, study, and use within flight simulation environments. They are **not a substitute** for official documentation and must **not** be used for the operation of real aircraft.

Before flying any real aircraft, you must:

- Use only the **Pilot's Operating Handbook (POH)** or **Airplane Flight Manual (AFM)** provided by the aircraft manufacturer.
- Comply with all applicable **aviation regulations** (FAA, EASA, or your regional authority).

The authors and contributors of this repository accept **no responsibility or liability** for any use of these checklists in real-world aviation. Always defer to official, manufacturer-approved materials.
